# NumVerifyPythonFlaskDemo


![](pythonflask.png)

## Create Environment
`py -m venv env`

## Activate Environment
`env\Scripts\activate`
`source env/bin/activate`

## Install Flask & Set Flask App
`pip install flask`
`pip install requests`
`set FLASK_APP=app.py`
